Combine Data Sources

Easy Insight has two ways that you can combine your data sources. Choose the way you would like to combine them below.

Join Different Sources

Do you want to join different data sources? For example, if you want to combine Basecamp data with Highrise data or FreshBooks data with spreadsheet data, you would use this option.

Join Federated, Similar Sources

Do you want to combine data sources of the same type that are federated across multiple sources? For example, if you want to combine multiple Quickbase instances or Google Spreadsheets into a single consolidated source, you would use this option. For now, if this is necessary, please use the full interface.
