Want to Explore Free Data?

Easy Insight offers users the ability to publish and distribute free data. If you want to simply explore some of Easy Insight's functionality against some existing data, the Data Exchange is a great place to start!

Immediate Connections to Data Sources

You can quickly pull data into Easy Insight for any of the data sources listed below. Simply click on the data source name to jump into the Easy Insight application and start running! Prebuilt reports are available with each solution to help improve your time to results. Further reports produced by the community can be found at the Solution Exchange.

  • Google Analytics
  • MySQL
  • Atlassian JIRA
  • Highrise
  • Salesforce.com
  • Basecamp
  • JetBrains Teamcity
  • SQL Server
