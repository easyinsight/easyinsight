Collaboration Documentation

Need help using collaboration in Easy Insight? Here's the documentation to guide you through the process.

How do I add users to my account?

You can add other users from the User Administration page under Account.

What are the concepts of collaboration inside of Easy Insight?

There are three levels of access inside of Easy Insight for data sources, reports, dashboards, scorecards and KPI trees:

  • Owner -- the person who actually created the object
  • Account Visible -- flag available when you save a data source
  • Group Visible -- the object has been associated through a group

For smaller teams, you can probably get away with doing but Owner and Account Visible settings. The Group Visible settings are more relevant to larger teams where you want to restrict certain sets of users to certain data sets.

How do I set access levels?

When you save a data source, report, dashboard, scorecard, or KPI tree, you have the option of setting a value for "Do you want to share this report with other users in the account?" If you check this option, the saved object will be visible to everyone in the account under My Data.

You can add users to a group by clicking in the group from the Group page, then clicking on Members and Add User.

How do I add users to a group?

You can add users to a group by clicking in the group from the Group page, then clicking on Members and Add User.

How do I add data sources and reports to groups?

Click to Data Sources and Reports under a given group and use the buttons on the right side of the page to add data sources or reports. These data sources and reports will become available to users in the group in both the group page and the individual user My Data pages.
