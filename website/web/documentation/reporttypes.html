How do I format a given field on a List or Tree report?

From the field editor, click on the Text tab. You can change Align to Left, Center, or Right. You can assign the column a fixed width. You can set the field to word wrap.

How do I customize the styling of a List or Tree report?

You can adjust the following styling options for a List or Tree report through Edit Report Properties/Additional Properties:

  • Alternating Row Color 1 and Alternating Row Color 2

    The background striping on the list or tree. For best visual appearance, these colors should be similar.

  • Fixed Report Width

    You can use this field to force a report to have a particular width. For example, if you have a large number of columns, you might use this field to force the report to display at a greater width than the default screen.

  • Font Size

    Changes the size of the font displayed in the report.

  • Header Top Color/Header Bottom Color/Header Text Color

    Styling around the background and text of the header section of the report.

  • Show Line Numbers

    Adds an additional line number column to the report (List only).

  • Summary Row

    Adds a summary row to the bottom of the report containing aggregate totals of all measures contained within the report (List only).

  • Summary Row Background Color/Text Color

    Background color and text color for the summary row.

  • Text Color

    Text Color is the actual color of the report text. It should be something highly contrasting from the Alternating Row colors.

How do I limit the number of results shown in a report?

You can set a limit to the number of results shown in a List or Tree report through Edit Report Properties/Limits.

How do I customize the styling of a Bar or Column chart?

You can adjust the following styling options for a Bar or Column report through Edit Report Properties/Additional Properties:

  • Chart Sort

    Applies a sort to the data based on the x-axis or y-axis.

  • Custom Chart Color/Custom Chart Gradient/Use Custom Chart Color

    If you toggle the "Use Custom Chart Color" option, the chart will use the Custom Chart Color and Custom Chart Gradient colors instead of the default color scheme.
