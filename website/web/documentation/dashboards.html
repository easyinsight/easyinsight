Dashboard Documentation

Need help using dashboards in Easy Insight? Here's the documentation to guide you through the process.

Table of Contents

  • What are the main dashboard pages?
  • How do I create a new dashboard?

What are the main dashboard pages?

There are two main pages for dashboards, the Dashboard View and the Dashboard Editor. The Dashboard View is a simplified view of the dashboard. It shows the dashboard itself, any filters, and some basic functionality for exporting the dashboard and its reports to other formats. The Dashboard Editor is a much more sophisticated screen for modifying or building new dashboards.

How do I create a new dashboard?

You can create new dashboards from your Home page. If you have a single data source on your system, you'll automatically start off on that particular data source. If you have multiple data sources, you'll need to click on a particular data source from the available list. Click on the New Dashboard button and you'll navigate into the dashboard editor.

Once in the dashboard editor, you'll be prompted to define the initial dashboard dimensions. Within these initial dimensions, you can add Grids, Stacks, and Reports to the dashboard. Grids can be any combination of rows and columns, so you can do 2x2, 2x1, 3x1, 3x2, etc. Stacks enable you to combine multiple elements into a single page, with either a button box or a dropdown box to toggle between panes. Finally, you add reports as child elements beneath everything else.
